## Congruence

Two integers are congruent modulo `n` when they have the same remainder after division by `n`.

```text
a ≡ b (mod n)
```

This is equivalent to saying that `n` divides `a - b`.

## Arithmetic rules

If `a ≡ b (mod n)` and `c ≡ d (mod n)`, then:

- `a + c ≡ b + d (mod n)`
- `a - c ≡ b - d (mod n)`
- `ac ≡ bd (mod n)`

Division requires more care because a multiplicative inverse may not exist.
