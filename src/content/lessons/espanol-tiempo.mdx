## 表達日期
問
- ¿Qué fecha es hoy?
- ¿Qué día es hoy?

答
- Es (ser - el/ella/usted) + `星期` (Es viernes.)
- Hoy es + `星期` + `日期` (Hoy es domingo 3.)
- Hoy es + `日期` + de + `月份` (Hoy es 9 de mayo.)
- Hoy es + `日期` + de + `月份` + de + `年` (Hoy es 1 de junio de 2014.)
- Hoy es + `星期` + `日期` + de + `月份` + de + `年` (Hoy es martes 27 de marzo de 2024.)

## 表達幾點幾分
問
- ¿Qué hora es?

答
- Es la una (en punto).
- Son las seis (en punto).
- Son las ocho y diez.
- Son las once y treinta/media.

<Callout title="補充" type="note">
  - en punto 是「整」的意思
  - 一點 (una) 要用 es、其他要用 son
</Callout>

後面可以加在一天當中的哪個時段：
|  | 時段 |
| - | - |
| la mañana | 早上 |
| el mediodía | 中午 |
| la tarde | 下午 |
| la noche | 晚上 |
| la medianoche | 半夜 |

<Callout title="補充" type="note">
  - 只說在什麼時段的時候介系詞用「en」或「por」
    - Tomo un café por/en la mañana.
  - 加上幾點幾分的時候用「de」
    - Tomo un café a las 8 de la mañana.
    - 如果是 de el mediodía 的話 de el 要縮寫變成 del mediodía
</Callout>
