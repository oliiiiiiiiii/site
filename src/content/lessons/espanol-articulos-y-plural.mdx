## los artículos
|  | 陰性 | 陽性 |
| - | - | - |
| 一個 | una | un |
| 一些 | unas | unos |
| 這個/那個 | la | el |
| 這些/那些 | las | los |
## plural
- 字尾母音：加 s
- 字尾子音：加 es

<Callout title="補充" type="note">
  形容詞也要跟著名詞變成複數。
  chocolate caliente $\rightarrow$ chocolates caliente**s**
</Callout>
