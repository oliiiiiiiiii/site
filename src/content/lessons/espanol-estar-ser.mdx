## 變化
|  | estar | ser |
| - | - | - |
| yo | estoy | soy |
| tú | estás | eres |
| él / ella / vd. | está | es |
| nosotros / nosotras | estamos | somos |
| vosotros / vosotras | estáis | sois |
| ellos / ellas / vds. | están | son |

## 用法
| estar | ser |
| - | - |
| 表狀態或地點 <br /> Estoy cansada. <br /> Estoy en el cine. | 表職業或身份 <br /> Soy profesor. <br /> Soy taiwanesa. |
